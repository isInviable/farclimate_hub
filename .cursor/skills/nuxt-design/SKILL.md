# Nuxt Design

Propose UI layout, component hierarchy, and UX. **Design only** — no implementation. Implementation uses the nuxt-ui-frontend skill.

## Output

- Layout and component breakdown (atoms/molecules/organisms).
- Which Nuxt UI components to use (UCard, UTable, UModal, etc.) and Tailwind approach.
- Responsive behavior (mobile-first, breakpoints).
- Accessibility and semantic structure (headings, labels, ARIA where needed).
- Optional: short copy or state descriptions.

## Guardrails

- Follow existing project patterns and `.cursor/rules/nuxt-vue3-components.mdc`.
- After approval, suggest: “Use nuxt-ui-frontend (or /frontend) to implement this.”
